# 外交部国家和组织离线信息库

这是一个本地 `Windows` 桌面程序，用于整理和离线检索外交部官网公开页面中的国家、地区和国际组织简介信息。

这个公开仓库提供源码；推荐的发布形态是：`源码仓库 + 不内置数据库的 exe + 单独发布的数据快照`。

## 公开发布说明

- 本项目是个人开发工具，`不是官方产品`。
- 仓库不包含抓取后的数据库，也不默认分发内置整库的 `exe`。
- 数据来源于外交部官网公开页面，具体内容、表述和更新情况以原站为准。
- 本项目主要用于学习、检索和信息整理，请在实际使用前自行评估数据合规性和适用场景。

## 当前功能

- 本地 SQLite 数据库存储
- 名称优先的离线检索与筛选
- “结构化信息 + 正文”并排阅读
- HTML 正文渲染和表格显示
- 世界地图独立窗口，点击国家后跳转正文
- 首次无数据时优先导入数据快照
- 保留“同步官网”作为备用更新入口

## 首次使用

1. 运行程序。
2. 首次启动会自动释放地图资源到：
   `%LOCALAPPDATA%\\MfaLocalDb`
3. 如果本地数据库为空，优先点击界面上的 `导入快照` 按钮，选择你发布的数据快照文件。
4. 只有在快照不可用或需要手动重建数据时，再使用 `同步官网`。

## 数据快照命名建议

建议把数据提取日期直接写进文件名，方便用户判断数据时效：

- `mfa-local-snapshot-2026-07-05.db`
- `mfa-local-snapshot-2026-07-05.zip`

如果你发布的是压缩包，建议在 Release 文案里同时注明：

- 快照提取日期
- 条目总数
- 是否包含国家与组织两类数据

## 开发运行

```powershell
dotnet run --project .\MfaLocalDb\
```

## 发布本地 exe

如果你只是给自己或受控范围内的同事使用，可以在本地自行构建：

```powershell
dotnet publish .\MfaLocalDb\ -c Release
```

发布产物位于：

`MfaLocalDb\bin\Release\net8.0-windows\win-x64\publish\MfaLocalDb.exe`

## 建议的公开分享方式

- GitHub 仓库放源码
- GitHub Releases 放 `不内置数据库` 的 exe
- GitHub Releases 单独放数据快照文件
- Release 文案明确说明快照日期、条目数量和数据来源
