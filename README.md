# 外交部国家和组织离线信息库

这是一个本地 `Windows` 桌面程序，用于整理和离线检索外交部官网公开页面中的国家、地区和国际组织简介信息。

这个公开仓库提供源码；推荐的发布形态是：`源码仓库 + 内置离线快照的 exe`。如果后续需要更新，也可以额外提供独立的数据快照文件。

## 公开发布说明

- 本项目是个人开发工具，`不是官方产品`。
- 发布版 exe 默认内置一份离线数据快照，方便开箱即用。
- 数据来源于外交部官网公开页面，具体内容、表述和更新情况以原站为准。
- 本项目主要用于学习、检索和信息整理，请在实际使用前自行评估数据合规性和适用场景。

## 当前功能

- 本地 SQLite 数据库存储
- 名称优先的离线检索与筛选
- “结构化信息 + 正文”并排阅读
- HTML 正文渲染和表格显示
- 世界地图独立窗口，点击国家后跳转正文
- 发布版默认内置离线快照，可直接使用
- 支持导入数据快照进行更新或修复
- 保留“同步官网”作为手动更新入口

## 首次使用

1. 运行程序。
2. 首次启动会自动释放内置数据库和地图资源到：
   `%LOCALAPPDATA%\\MfaLocalDb`
3. 默认情况下可直接离线检索。
4. 如果需要切换到新快照，可点击 `导入快照`。
5. 如果需要自行重建数据，可使用 `同步官网`。

## 数据快照命名建议

建议把数据提取日期直接写进文件名，方便用户判断数据时效：

- `mfa-local-snapshot-2026-07-05.db`
- `mfa-local-snapshot-2026-07-05.zip`

## 开发运行

```powershell
dotnet run --project .\MfaLocalDb\
```

## 发布本地 exe

```powershell
dotnet publish .\MfaLocalDb\ -c Release
```

发布产物位于：

`MfaLocalDb\bin\Release\net8.0-windows\win-x64\publish\MfaLocalDb.exe`

## 建议的公开分享方式

- GitHub 仓库放源码
- GitHub Releases 放 `内置离线快照` 的 exe
- Release 文案明确说明快照日期、条目数量、数据来源和“非官方产品”说明
