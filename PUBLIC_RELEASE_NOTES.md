# Public Release Notes

这个仓库已经调整为更适合公开分享的形态：

- 发布版 exe 内置一份离线数据快照，方便开箱即用
- 仍保留 `导入快照`，用于更新或修复本地数据库
- 仍保留 `同步官网`，用于手动重建数据
- 仓库本身不跟踪测试 exe 和临时发布目录

## 推荐发布结构

1. GitHub 仓库：源码
2. GitHub Release 附件一：内置离线快照的 exe
3. 可选附件：独立数据快照（给需要手动替换的人）

## 快照命名建议

- `mfa-local-snapshot-2026-07-05.db`
- `mfa-local-snapshot-2026-07-05.zip`

建议把提取日期直接写进文件名，并在 Release 里补充：

- 快照提取日期
- 条目总数
- 数据来源说明
- 非官方产品说明
- 具体内容与更新以原站为准

## 发布前检查

1. 默认启动时能直接看到离线内容。
2. `导入快照` 可以成功导入外部 `.db` 文件。
3. `同步官网` 仍能作为备用入口正常工作。
